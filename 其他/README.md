AI-Assisted Cyber Risk Assessment System

AI 輔助資訊資產風險評估與建議系統

Project Overview

本專案旨在設計一套結合 生成式 AI 與規則引擎的資安風險管理輔助系統，協助中小型電商企業完成資訊資產的風險評估流程。

系統透過 資產清冊分析、風險分數計算與 AI 建議，協助企業理解資產的重要性並提供對應的安全控制建議。

本研究重點在於提升風險評估流程的 標準化、可解釋性與自動化程度。

Problem Statement

許多中小型企業在實施資訊安全管理系統（ISMS）時，常面臨以下問題：

資產盤點後，CIA 評估高度依賴人工經驗

不同部門對資產重要性的判斷標準不一致

風險評估通常為年度作業，缺乏即時更新能力

即使得到風險分數，管理者仍難以理解應採取的安全措施

ISO 27001 等資安框架內容複雜，難以直接應用

因此，本專案設計一套 AI 輔助系統來改善上述問題。
